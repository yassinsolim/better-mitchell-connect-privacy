Privacy Policy

Better Mitchell Connect

Effective date: July 20, 2026. Selecting a PDF sends extracted text through the access-controlled Better Mitchell Connect service to OpenAI for structured field extraction. PDF bytes are not transmitted.

Information handled

The extension handles PDFs selected explicitly by the user, the visible Mitchell Connect Create New Job form, and rendered Mitchell Estimating rows and controls used for reviewed draft staging. A claim PDF may contain names, postal addresses, email addresses, phone numbers, claim and policy identifiers, vehicle identifiers, adjuster information, collision details, parts, labor, and pricing.

How information is used

Information is used only to extract structured claim and estimate fields, fill the currently visible Mitchell Contact, Vehicle, or Insurance fields, and review and stage uncommitted estimate rows after an explicit user action. The beta never activates Mitchell Commit or Mark as Final.

Storage, transmission, and retention

  • PDF bytes remain in memory in the active browser tab and are not sent to OpenAI.
  • Selecting a PDF sends extracted text through the publisher-operated Better Mitchell Connect service in Google Cloud Run to OpenAI with application storage disabled for the request.
  • OpenAI states that API data is not used for training unless the API organization opts in.
  • OpenAI processing remains subject to the publisher's OpenAI project settings, agreement, and applicable data policies.
  • The publisher service transiently processes text and parsed results but does not intentionally log request bodies or persist claim content. Infrastructure logs may retain request metadata such as timestamp, status, and latency.
  • The extension has no analytics, advertising, or remote OCR.
  • The extension uses memory-backed chrome.storage.session for claim drafts. chrome.storage.local stores only the revocable beta access code until the user removes it or uninstalls the extension.
  • Closing the tab, restarting the browser, or reloading the extension clears the session draft.

Sharing and sale

The publisher does not sell or permit human review of extension user data. The publisher-operated service and OpenAI process extracted text only to provide field extraction after the user selects a PDF. Data is not used for advertising, profiling, creditworthiness, or lending.

Chrome Web Store Limited Use

Use of information received from Chrome APIs and the Mitchell Connect page adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements. Data is used only to provide the user-facing PDF review and form-filling feature.

User control

Users choose whether to select a PDF, may edit every detected value before filling, and retain control over Mitchell's Save & Next action. Uninstalling the extension removes its access.
