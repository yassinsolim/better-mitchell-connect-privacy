Privacy Policy

Better Mitchell Connect

Effective date: July 19, 2026. Selecting a PDF sends extracted text to OpenAI for structured field extraction. PDF bytes are not transmitted, and the publisher does not receive the text.

Information handled

The extension handles PDFs selected explicitly by the user, the visible Mitchell Connect Create New Job form, and rendered Mitchell Estimating rows used for read-only comparison. A claim PDF may contain names, postal addresses, email addresses, phone numbers, claim and policy identifiers, vehicle identifiers, adjuster information, collision details, parts, labor, and pricing.

How information is used

Information is used only to extract structured claim and estimate fields, fill the currently visible Mitchell Contact, Vehicle, or Insurance fields, and compare parsed estimate rows read-only. The beta does not add, commit, or finalize estimate lines.

Storage, transmission, and retention

  • PDF bytes remain in memory in the active browser tab and are not sent to OpenAI.
  • Selecting a PDF sends extracted text through a user-operated local service to OpenAI with application storage disabled for the request.
  • OpenAI states that API data is not used for training unless the API organization opts in.
  • OpenAI processing remains subject to the user's OpenAI account settings, agreement, and applicable data policies.
  • The extension has no publisher-operated backend, analytics, advertising, or remote OCR.
  • The extension uses memory-backed chrome.storage.session for the current tab and does not use persistent local or sync storage.
  • Closing the tab, restarting the browser, or reloading the extension clears the session draft.

Sharing and sale

The publisher does not sell, receive, or permit human review of extension user data. Transfer to OpenAI occurs only to provide field extraction after the user selects a PDF. Data is not used for advertising, profiling, creditworthiness, or lending.

Chrome Web Store Limited Use

Use of information received from Chrome APIs and the Mitchell Connect page adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements. Data is used only to provide the user-facing PDF review and form-filling feature.

User control

Users choose whether to select a PDF, may edit every detected value before filling, and retain control over Mitchell's Save & Next action. Uninstalling the extension removes its access.
